User

Represents an user in the system.

Attributes

  • name: user name (string)
  • turnsPlayed: total amount of turns across all games (integer)
  • totalTime: time in seconds spent making turns across all games (integer)
  • games: list of game instances the user has participated in (string)

Game type

Represents a playable game type in the system.

Attributes

  • name: name of the game type (string)
  • defaultState: the starting state for a specific game type (string)

Game

Represents a game instance in the system.

Attributes

  • id: UUID used to identify game instances (string)
  • type: the type of game of a given game instance (string)
  • result: result of the game after the latest move:
    • -1: game is not finished
    • 0: game ended in a draw
    • 1: team 1 won the game
    • 2: team 2 won the game
  • state: current state of the game instance. Format depends on the game type (string)
  • currentPlayer: name of the player currently playing the game instance (string)
  • moveHistory: list of moves made in the game instance. Format depends on the game type (string)
  • players: list of users that have participated in the game instance (string)
