boardgame:games-all

Leads to the collection resource of all game instances.

boardgame:users-all

Leads to the collection resource of all registered users.

boardgame:gametypes-all

Leads to the collection resource of all playable game types.

boardgame:add-game

Contains the URL and JSON schema used to create a new game instance. Requires admin authentication.

boardgame:add-user

Contains the URL and JSON schema used to create a new user.

boardgame:add-gametype

Contains the URL and JSON schema used to create a new game type. Requires admin authentication.

boardgame:get-random

Leads to a resource used for GETting a random empty game instance of a given game type.

boardgame:join-game

Leads to a resource used for setting the current user to be the current player of a given game instance.

boardgame:make-move

Contains instructions on how to make a move in a given game instance.

boardgame:delete

Used for deleting a game instance, user or game type. Required authentication varies.
